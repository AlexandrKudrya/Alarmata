# core.di

Hilt dependency injection modules.

## Files
- `AppModule.kt` — Provides application-level dependencies (Context, shared preferences, etc.).
- `DatabaseModule.kt` — Provides Room database instance and DAOs.
- `NetworkModule.kt` — Provides Retrofit, OkHttp client, and API services.
- `RepositoryModule.kt` — Binds repository interfaces to their implementations.
