# core

Core infrastructure shared across the Sleep Guardian application.

## Subfolders
- `di/` — Hilt dependency injection modules for providing app-wide dependencies.
- `database/` — Room database setup, converters, and migrations.
- `alarm/` — AlarmManager scheduling and triggering infrastructure.
- `notifications/` — Notification management and channel creation.
- `utils/` — Utility functions, extension functions, and helpers.
- `constants/` — App-wide constants and configuration values.
