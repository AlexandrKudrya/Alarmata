# data.local.dao

Room Data Access Objects.

## Files
- `AlarmDao.kt` — DAO for alarm CRUD operations and queries.
- `AlarmLogDao.kt` — DAO for alarm log entries (dismissal history).
- `AchievementDao.kt` — DAO for achievement/badge data.
- `BedtimeDao.kt` — DAO for bedtime checklist and routine data.
- `SleepSessionDao.kt` — DAO for sleep session tracking data.
