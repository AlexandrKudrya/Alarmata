# data.local.entities

Room entity classes representing database tables.

## Files
- `AlarmEntity.kt` — Entity for alarms table storing alarm configuration.
- `AlarmLogEntity.kt` — Entity for alarm_logs table tracking dismissal events.
- `AchievementEntity.kt` — Entity for achievements table storing unlocked badges.
- `SleepSessionEntity.kt` — Entity for sleep_sessions table tracking sleep data.
- `BedtimeChecklistEntity.kt` — Entity for bedtime checklist items.
- `BedtimeLogEntity.kt` — Entity for bedtime routine completion logs.
- `AlarmProfileEntity.kt` — Entity for alarm profiles with preset configurations.
