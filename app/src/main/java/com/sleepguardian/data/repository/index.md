# data.repository

Repository interface implementations.

## Files
- `AlarmRepositoryImpl.kt` — Implementation of AlarmRepository coordinating local storage and alarm scheduling.
- `StatisticsRepositoryImpl.kt` — Implementation of StatisticsRepository aggregating sleep data and analytics.
- `TaskRepositoryImpl.kt` — Implementation of TaskRepository handling task generation (local and AI-based).
- `BedtimeRepositoryImpl.kt` — Implementation of BedtimeRepository managing bedtime routines and checklists.
