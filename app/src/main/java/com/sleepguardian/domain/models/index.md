# domain.models

Domain model classes representing core business entities.

## Files
- `Alarm.kt` — Alarm domain model with time, repeat pattern, tasks, and settings.
- `AlarmLog.kt` — Log entry recording alarm dismissal events and completion data.
- `Task.kt` — Task model representing wake-up tasks (type, difficulty, configuration).
- `Achievement.kt` — Achievement/badge model for user accomplishments.
- `SleepSession.kt` — Sleep session model tracking bedtime, wake time, and quality.
- `BedtimeChecklist.kt` — Bedtime routine checklist item model.
