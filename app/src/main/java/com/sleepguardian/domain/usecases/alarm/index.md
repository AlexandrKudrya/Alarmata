# domain.usecases.alarm

Alarm-related use cases.

## Files
- `CreateAlarmUseCase.kt` — Creates a new alarm and schedules it.
- `DeleteAlarmUseCase.kt` — Deletes an alarm and cancels its scheduling.
- `UpdateAlarmUseCase.kt` — Updates an existing alarm and reschedules if needed.
- `GetAlarmsUseCase.kt` — Retrieves all alarms or a specific alarm.
- `ToggleAlarmUseCase.kt` — Enables or disables an alarm.
