# features.active_alarm

Active alarm screen displayed when alarm triggers.

## Files
- `ActiveAlarmActivity.kt` — Separate Activity with wake lock flags and full-screen display.

## Subfolders
- `ui/` — ActiveAlarmScreen and task view components (MathTaskView, QRScannerView, ShakeTaskView, etc.).
- `viewmodel/` — ActiveAlarmViewModel managing alarm state and task validation.
