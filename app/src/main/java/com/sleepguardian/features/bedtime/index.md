# features.bedtime

Bedtime mode with evening rituals, breathing exercises, and sleep sounds.

## Subfolders
- `ui/` — BedtimeModeScreen, BreathingExerciseScreen, SleepSoundsScreen composables.
- `viewmodel/` — BedtimeViewModel managing bedtime routine state.
- `service/` — SleepSoundsService for playing ambient sounds in background.
