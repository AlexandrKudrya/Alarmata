# features

Feature modules organized by functionality (presentation layer).

## Subfolders
- `alarms/` — Alarm list and CRUD screens for managing alarms.
- `active_alarm/` — Active alarm screen displayed when an alarm triggers.
- `tasks/` — Task engine and task type implementations.
- `statistics/` — Sleep statistics, charts, and analytics screens.
- `sleep_cycles/` — Sleep cycle calculator feature.
- `bedtime/` — Bedtime mode with routines, breathing exercises, and sleep sounds.
- `achievements/` — Achievements and streaks display.
- `settings/` — App settings and preferences.
