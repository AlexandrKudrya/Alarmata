# com.sleepguardian

Root package for the Sleep Guardian Android application.

## Subfolders
- `app/` — Application entry points and main activity.
- `core/` — Core infrastructure shared across the app (DI, database, alarm scheduling, notifications, utilities).
- `domain/` — Domain layer with business logic (models, repository interfaces, use cases).
- `data/` — Data layer implementations (local/remote data sources, repository implementations).
- `features/` — Feature modules organized by functionality (presentation layer with UI and ViewModels).
- `presentation/` — Shared presentation resources (theme, navigation).
- `widget/` — Home screen widget implementation.
