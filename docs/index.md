# docs

Documentation for the Sleep Guardian project.

## Files
- `README.md` — Project overview, features, and setup instructions.
- `ROADMAP.md` — Development roadmap with planned features and milestones.
- `ARCHITECTURE.md` — Technical architecture documentation (Clean Architecture + MVVM).
- `CI-CD.md` — CI/CD pipeline setup and deployment documentation.
- `CODESTYLE.md` — Code style guide and conventions for the project.
- `RULES.md` — AI development rules and guidelines for working with AI assistants.
- `MANUAL-TASKS.md` — List of tasks requiring human intervention (API keys, Play Store setup, etc.).

## Subfolders
- `iterations/` — Per-iteration development plans and progress documentation.
