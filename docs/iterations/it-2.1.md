# Iteration 2.1: Advanced Math Tasks

**Итерация:** 2 (Умные задания)
**Срок:** 4-5 дней
**Статус:** Не начата

## Цель

Расширить математические задания новыми типами задач, чтобы они не становились скучными и рутинными.

## Задачи

- [ ] Новые типы математических задач:
  - Смешанные операции: `(17 * 3) + 48 / 6`
  - Числовые последовательности: `2, 4, 8, 16, ?`
  - Простые уравнения: `2x + 5 = 17, найти x`
- [ ] Генераторы для каждого типа:
  - `MixedOperationGenerator`
  - `SequenceGenerator`
  - `EquationGenerator`
- [ ] Настройка сложности для каждого типа:
  - Easy: однозначные числа, простые паттерны
  - Medium: двузначные, 2 операции
  - Hard: трёхзначные, 3+ операций, скобки
- [ ] Рандомизация типа задачи внутри категории "Math"
- [ ] UI для выбора подтипов (опционально)

## Примеры по типам

### Смешанные операции
- Easy: `5 + 3 * 2 = ?`
- Medium: `(34 + 17) * 3 = ?`
- Hard: `(47 * 3 + 28) / 13 = ?`

### Числовые последовательности
- Easy: `2, 4, 6, 8, ?` (арифметическая)
- Medium: `3, 6, 12, 24, ?` (геометрическая)
- Hard: `1, 1, 2, 3, 5, ?` (Фибоначчи)

### Уравнения
- Easy: `x + 5 = 12`
- Medium: `2x + 5 = 17`
- Hard: `3x - 7 = 2x + 5`

## Критерии завершения

- 3+ подтипа математических задач
- Каждый подтип имеет 3 уровня сложности
- Рандомизация работает — задачи не повторяются
- Генерация корректна (ответы всегда целые числа)

## Зависимости

- MVP 1.4 (Task Engine, MathTask базовый)
