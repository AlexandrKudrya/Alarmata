# Iteration 2.2: New Task Types + Combinations

**Итерация:** 2 (Умные задания)
**Срок:** 5-7 дней
**Статус:** Не начата

## Цель

Добавить новые типы заданий (Memory Game, Typing Challenge, Multiple QR) и возможность комбинировать задания.

## Задачи

### Memory Game Task
- [ ] Генерация последовательности 4-6 цветов/символов
- [ ] Показ последовательности на 3-5 секунд
- [ ] Воспроизведение пользователем (нажатие на цветные кнопки)
- [ ] Визуализация: grid цветных кнопок с анимацией
- [ ] Easy: 4 элемента, Medium: 5, Hard: 6

### Typing Challenge Task
- [ ] База мотивационных фраз (50+ фраз)
- [ ] Отображение фразы, ввод пользователем
- [ ] Детекция опечаток в реальном времени (highlight ошибок)
- [ ] Easy: короткие фразы (3-5 слов), Medium: 5-8 слов, Hard: 8-12 слов
- [ ] Таймер (опционально)

### Multiple QR Codes Task
- [ ] Настройка 2-3 QR-кодов при создании будильника
- [ ] Последовательное сканирование при пробуждении
- [ ] UI: прогресс "Отсканировано 2/3"
- [ ] Управление списком кодов (добавить, удалить, пересканировать)

### Комбинированные задания
- [ ] Настройка: Sequential ("Math -> QR") или Choice ("Math OR Shake")
- [ ] Sequential: задания выполняются последовательно
- [ ] Choice: случайно выбирается одно из списка
- [ ] UI для конфигурации комбинаций в AddEditAlarmScreen

## Критерии завершения

- 5+ типов заданий работают (Math, QR, Shake, Memory, Typing)
- Комбинации заданий настраиваются
- Каждое задание протестировано на всех уровнях сложности
- UI интуитивен для каждого типа

## Зависимости

- IT-2.1 (расширенный Task Engine)
- MVP 1.5 (базовые QR и Shake задания)
