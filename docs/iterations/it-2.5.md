# Iteration 2.5: UI/UX Polish & Design System

**Итерация:** 2.5 (между заданиями и аналитикой)
**Срок:** 7-10 дней
**Статус:** Не начата

## Цель

Привести весь UI в единый стиль, создать дизайн-систему, добавить анимации и микро-взаимодействия. После этой итерации приложение должно выглядеть как продакшн-продукт, а не прототип.

## Почему здесь

К этому моменту готовы:
- Все экраны будильника (CRUD, список, активный будильник)
- Все типы заданий (Math, QR, Shake, Memory, Typing, AI)
- Быстрый сон + циклы сна
- Адаптивная сложность
- Базовая статистика

Ещё не начаты: аналитика, вечерний режим, gamification — они получат дизайн сразу.

## Задачи

### Design System / Tokens

- [ ] Определить и зафиксировать цветовую палитру:
  - Primary, Secondary, Tertiary + on-variants
  - Light + Dark тема
  - Semantic colors: success, warning, error, sleep (ночной акцент)
- [ ] Типографика:
  - Display (часы на экране будильника)
  - Headlines (заголовки экранов)
  - Body (основной текст)
  - Label (чипсы, кнопки)
  - Mono (ввод кода, математика)
- [ ] Spacing/sizing tokens (padding, radius, elevation)
- [ ] Вынести всё в `Theme.kt`, `Color.kt`, `Type.kt`, `Shape.kt`

### App Icon & Branding

- [ ] Иконка приложения (заменить placeholder):
  - Adaptive icon для Android 8+
  - Foreground + background layers
  - Монохромная версия для Android 13+
- [ ] Splash screen (Android 12+ SplashScreen API)
- [ ] Подобрать шрифт (Google Fonts: Inter, Nunito, или Manrope)

### Экран списка будильников

- [ ] Красивые карточки:
  - Gradient или subtle background
  - Выразительная типографика для времени
  - Анимация toggle on/off
  - Swipe-to-delete с красной подложкой и иконкой
- [ ] Empty state: иллюстрация + текст (не просто текст)
- [ ] FAB с анимацией

### Экран редактирования будильника

- [ ] TimePicker: кастомный или стилизованный системный
- [ ] Day-of-week selector: pill-shaped, анимация выбора
- [ ] Секции с иконками и dividers
- [ ] Плавные transitions между секциями

### Экран активного будильника

- [ ] Полноэкранный дизайн — immersive
- [ ] Анимация: пульсирующая кнопка Dismiss
- [ ] Градиент фона (утренние тона: мягкий оранжевый → синий)
- [ ] Кнопки: Dismiss крупная, Snooze мелкая и неприметная (отговариваем)
- [ ] Анимация перехода при выключении

### Экран заданий

- [ ] Progress bar с анимацией
- [ ] Зелёная/красная вспышка при правильном/неправильном ответе
- [ ] Shake: анимация тряски телефона
- [ ] Memory: плавные переходы карточек
- [ ] Тёмный фон (человек только проснулся — яркость минимальна)

### Quick Sleep экран

- [ ] Ночная тема (глубокий синий/фиолетовый)
- [ ] Слайдер с haptic feedback
- [ ] Анимация луны/звёзд (Lottie или Compose Canvas)
- [ ] Карточки вариантов с мягкими тенями

### Общие компоненты

- [ ] Кастомные анимации переходов между экранами
- [ ] Shimmer loading states
- [ ] Snackbar стилизация
- [ ] Bottom sheet для быстрых действий
- [ ] Haptic feedback на ключевых действиях

### Accessibility

- [ ] Content descriptions для всех иконок
- [ ] Minimum touch target 48dp
- [ ] Contrast ratio >= 4.5:1
- [ ] TalkBack совместимость

### Dark Theme

- [ ] Полноценная тёмная тема (не просто инверсия)
- [ ] AMOLED black вариант (опционально в настройках)
- [ ] Тема по умолчанию = следовать системе

## Инструменты

- **Lottie** — для сложных анимаций (опционально)
- **Compose Animation API** — для transitions, shared element
- **Google Fonts** — типографика
- **Material You Dynamic Colors** — поддержка на Android 12+

## Критерии завершения

- Приложение выглядит как готовый продукт из Google Play
- Light + Dark тема полностью проработаны
- Анимации плавные, нет visual glitches
- Accessibility проверена
- Скриншоты для Google Play можно делать

## Зависимости

- Iteration 2.3 (все основные экраны готовы)
- MVP 1.6 (Quick Sleep экран)

## Риски

- Lottie анимации могут увеличить размер APK — использовать умеренно
- Custom шрифт может не поддерживать кириллицу — проверить заранее
- Чрезмерная стилизация может замедлить UI — профилировать
