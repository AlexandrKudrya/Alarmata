# Iteration 4.2: Sleep Timer + Breathing Exercises

**Итерация:** 4 (Вечерний режим + Ритуалы)
**Срок:** 4-5 дней
**Статус:** Не начата

## Цель

Помощь в засыпании: таймер сна и дыхательные упражнения с визуализацией.

## Задачи

### Sleep Timer
- [ ] Countdown timer: 15 / 20 / 30 / 45 / 60 минут (настраивается)
- [ ] Запуск из Bedtime Mode
- [ ] Gentle notification по окончании (мягкий звук)
- [ ] Автоматическая остановка звуков сна (если играют)

### Breathing Exercises
- [ ] Техники:
  - 4-7-8: вдох 4с → задержка 7с → выдох 8с
  - Box Breathing: 4-4-4-4
  - Relaxing Breath: 4-2-6
- [ ] Анимация: круг expand/contract синхронно с фазами
- [ ] Текстовые подсказки: "Вдох...", "Задержка...", "Выдох..."
- [ ] Haptic feedback синхронно с дыханием (мягкая вибрация)
- [ ] Таймер: количество циклов (5, 10, 15) или время (3, 5, 10 мин)
- [ ] Ambient звук (опционально): тихий белый шум на фоне

### UI
- [ ] Экран выбора техники дыхания
- [ ] Полноэкранная анимация дыхания (тёмная тема, минимум UI)
- [ ] Интеграция в Bedtime Mode (кнопка "Дыхательные упражнения")

## Критерии завершения

- Sleep timer работает и отправляет notification
- 3 техники дыхания реализованы
- Анимация плавная и помогает расслабиться
- Haptic feedback синхронен
